# expense-tracker:
a simple expense tracker for companies with file saving built in python.

# functionalities:
1- add as much expenses you want until it reaches the limit set by the user.
2- spending limit alert system.
3- final report with all the expenses, average and breakdown.
4- saves the full report in a '.txt' file.

# how to use:
1- make sure you have python installed.
2- run the program:
```bash
python expense_tracker.py
